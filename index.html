Step 1

Apa itu WhatsApp Business API?

WhatsApp Business API adalah layanan resmi dari Meta (Facebook) yang memungkinkan kamu mengirim pesan otomatis via WhatsApp menggunakan kode program (API).

❌ Cara Lama (Baileys/Unofficial)

  • Library tidak resmi
  • Rawan banned oleh WhatsApp
  • Perlu scan QR berulang
  • Sesi sering putus
  • Tidak ada support resmi

✅ Cara Baru (WA Business API)

  • Resmi dari Meta
  • Tidak bisa dibanned
  • Token-based, tidak perlu QR
  • 99.9% uptime stabil
  • Dokumentasi & support lengkap
⚠️ Keterbatasan: WA Business API tidak bisa kirim ke grup dan tidak bisa tag all. Hanya personal chat (1-on-1) via template yang sudah di-approve Meta.

Step 4

Setup WhatsApp Product

4.1

Tambahkan WhatsApp ke App

  1. Di dashboard app, scroll ke bawah
  2. Cari produk "WhatsApp"
  3. Klik "Set Up"
  4. Pilih Meta Business Account → Continue
4.2

Catat Credentials

Setelah setup, akan muncul halaman "API Setup" dengan info penting:

📋 Catat info ini:
Phone Number ID  : 123456789012345      ← untuk kirim pesan
WABA ID          : 987654321098765      ← untuk management
Access Token     : EAAGk0B3a...         ← untuk auth (sementara 24 jam)
⚠️ Token dari halaman ini hanya berlaku 24 jam. Kita akan buat token permanen di Step 8.
4.3

Daftarkan Nomor Penerima (Test Mode)

Selama masih test mode, kamu hanya bisa kirim ke nomor yang sudah didaftarkan:

  1. Di halaman API Setup, cari bagian "To"
  2. Klik "Manage phone number list"
  3. Klik "Add phone number"
  4. Masukkan nomor WA penerima: 6281234567890
  5. Penerima akan dapat kode verifikasi → masukkan kode
📝 Format nomor: 628xxx (pakai kode negara 62, tanpa + atau 0)
Step 5

Buat Message Template

WhatsApp Business API mengharuskan pesan dikirim dalam format template yang sudah di-approve Meta. Ini untuk mencegah spam.

🎉 Kabar baik: Meta sudah sediakan template bawaan hello_world yang langsung bisa dipakai untuk testing!
5.1

Buat Template Custom (Opsional)

Jika ingin template sendiri:

  1. Buka WhatsApp Manager → Message Templates
  2. Atau: Buka langsung
  3. Klik "Create Template"

7.3

Buat file send.js

send.js — Kirim pesan pertama via Node.js
require('dotenv').config();
const https = require('https');

const PHONE_ID = process.env.WA_PHONE_NUMBER_ID;
const TOKEN    = process.env.WA_ACCESS_TOKEN;
const VERSION  = process.env.WA_API_VERSION || 'v21.0';

function kirimPesan(nomorTujuan, templateName, params = [], lang = 'en_US') {
  return new Promise((resolve, reject) => {
    const body = JSON.stringify({
      messaging_product: 'whatsapp',
      to: nomorTujuan,
      type: 'template',
      template: {
        name: templateName,
        language: { code: lang },
        components: params.length ? [{
          type: 'body',
          parameters: params.map(p => ({ type: 'text', text: String(p) }))
        }] : []
      }
    });

    const req = https.request({
      hostname: 'graph.facebook.com',
      path: `/${VERSION}/${PHONE_ID}/messages`,
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${TOKEN}`,
        'Content-Type': 'application/json'
      }
    }, (res) => {
      let data = '';
      res.on('data', c => data += c);
      res.on('end', () => {
        const json = JSON.parse(data);
        if (res.statusCode === 200) {
          console.log('✅ Pesan terkirim!');
          console.log('   Message ID:', json.messages?.[0]?.id);
          resolve(json);
        } else {
          console.error('❌ Gagal:', json.error?.message);
          reject(new Error(json.error?.message));
        }
      });
    });
    req.on('error', reject);
    req.write(body);
    req.end();
  });
}

// ═══ JALANKAN ═══
async function main() {
  console.log('🚀 Mengirim pesan via WA Business API...\n');

  // Ganti nomor ini dengan nomor tujuan kamu
  await kirimPesan('6281234567890', 'hello_world');
}

main().catch(console.error);

Step 8

Buat Token Permanen

Token dari dashboard hanya 24 jam. Untuk production, buat System User Token yang tidak pernah expire:

  1. Buka business.facebook.com/settings/system-users
  2. Klik "Add" → nama: wa-bot, role: Admin
  3. Klik user → "Add Assets" → pilih Apps → centang app WA → Full Control
  4. Klik "Generate New Token"
  5. Pilih app → centang: whatsapp_business_messaging + whatsapp_business_management
  6. Klik "Generate Token" → COPY & SIMPAN!
  7. Update .env dengan token baru
🔐 RAHASIA! Token ini seperti password. Jangan share atau commit ke GitHub.

ErrorPenyebabSolusi
Recipient not in allowed listNomor belum terdaftar (test mode)Tambahkan nomor di API Setup

Access token expiredToken sementara (24 jam)Buat permanent token (Step 8)
Invalid parameterFormat nomor salahPakai 628xxx tanpa + atau 0
Message undeliverableNomor tidak punya WAPastikan nomor aktif di WhatsApp

Format Nomor yang Benar

✅ Benar: 6281234567890   (kode negara 62 + nomor)
❌ Salah: +6281234567890  (jangan pakai +)
❌ Salah: 081234567890    (jangan pakai 0 di depan)
❌ Salah: 81234567890     (harus ada kode negara)
